What is CodeX

Being able to create something using code is one of the best experiences one will ever have no matter how basic the program. At CodeX, we aim to provide you with all the fundamental tools to get you started on your programming journey. We have programming definitions from different categories and programming careers and tutorials for you to browse.

This platform is open to all aspiring programmers. Whether you are a person with no programming experience or an experienced programmer who wants to brush up on the fundamentals of programming, this platform is for you. We encourage that you use this opportunity to to build a solid foundation for your programming career.
